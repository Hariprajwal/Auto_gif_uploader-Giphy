# 📤 Auto_gif_uploader-Giphy: Automated Distribution Framework

**Auto_gif_uploader-Giphy** is a robust Python utility designed to bridge the gap between local content production and global distribution. By leveraging the **GIPHY API**, it allows developers to automate the process of uploading, tagging, and organizing GIFs at scale.

## ✨ Features
* **Seamless API Integration:** Full support for GIPHY's upload endpoints.
* **Bulk Processing:** Upload hundreds of assets in a single execution.
* **Metadata Automation:** Programmatically assign tags, ratings, and source URLs.
* **Error Handling:** Robust logging to track upload status and API rate limits.

## 🛠 Tech Stack
* **Language:** Python 3.12
* **API Handler:** Requests / Giphy SDK
* **Security:** Dotenv for secure API Key management

## 🚀 Setup & Usage
1. Obtain your API Key from the [Giphy Developers Portal](https://developers.giphy.com/).
2. Create a `.env` file and add your key: `GIPHY_API_KEY=your_key_here`
3. Run the uploader:
   
   python main.py --folder ./my-gifs --tags "manhwa, viral, anime"
